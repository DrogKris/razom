# razom
platform to connect studens among events
this is a minimalistic  mini-app inspired by Base.
It allows users to browse events, view details, register locally, see their events in a calendar, and filter events.

Key Features:

Events Feed with large cards

Event Details page with "Find Company" feature

My Events page (registered events)

Calendar page with emoji indicators for registered events

Profile page (editable interests and info)

Local filtering by Time, Category, and Location
